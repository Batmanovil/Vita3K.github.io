About Vita3K

Vita3K is an experimental open-source PlayStation Vita emulator for Windows and macOS.

Download

Official releases will be provided on the GitHub releases page.

The latest builds for Windows can be downloaded on AppVeyor.

Community

You can get in touch with developers and users in the following ways:

  • Discord (preferred)
  • IRC #vita3k on freenode

Compatibility

The emulator does not yet run any commercial games.

Several homebrew games are supported. Please check the compatibility database for details!

Contribute

You can contribute to the emulator in the following ways:

  • File issues or make pull requests on our GitHub repo.
  • Update the Homebrew compatibility database.

Donate

If you would like to show your appreciation or even help fund development, the project has a Patreon page.
